# Quick Start Guide

## Prerequisites
- Node.js (v14+)
- MongoDB (local or Atlas)

## Installation (3 steps)

### 1. Install Dependencies
```bash
npm run install-all
```

### 2. Configure Database
Create `server/.env`:
```
PORT=5000
MONGODB_URI=mongodb://localhost:27017/ecom-cart
```

### 3. Start Application
```bash
npm run dev
```

## Access
- Frontend: http://localhost:3000
- Backend: http://localhost:5000

## What You Get
✅ 8 luxury products pre-loaded
✅ Shopping cart with add/remove/update
✅ Checkout with receipt
✅ Responsive design
✅ MongoDB persistence
